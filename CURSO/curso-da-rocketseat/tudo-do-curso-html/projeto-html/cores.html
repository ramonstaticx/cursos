Cores no fundo( background-color)

a tag background-color pode ser usada para por cores inteiras no fundo, ou degradê vamos aos ex:

Aqui usei cor maciça e logo abaixo o código.

            
                    <div style="background-color:rgb(0, 217, 255);">
                        ou no CSS
                        div {
                            background-color:rgb(0, 217, 255)
                        }

agora com fundo degradê da direita para esquerda, depois de baixo para cima (observe que vai dar um erro e para isso tenho de corrigir com a tag universal *)

Vamos substituir a tag "color" por "imagen", e para posicionar vamos usar o "to".

depois da tag "to" podemos usar right, left, top, bottom ou em graus para posicionar exemplo 45deg, "deg" significa graus

Outra coisa importante que pode ser mudada é o linear-gradient por radial-gradient, e acrescentar logo em seguida ao invés do " To" usar "circle".

após a cor posso por porcentagem para que tenha uma porcentagem a mais daquela cor na página.

Direita para esquerda 

            
            <div style="background-image:linear-gradient(to right,white, blue)"></div>

            no CSS

            body {
                background-image:linear-gradient(to right, white,blue);
            }

Agora de baixo para cima e observe o erro citado la em cima e como corrigir no CSS

de baixo para cima 

            
         
                
                     <div style="background-image:linear-gradient(to bottom, white,blue) ;">
              
                     no CSS
             
                     *{
             height: 100%
                     }
             /*tivemos de pôr esse valor por causa que
              o degradê fica picado, assim fica um todo */
                     
                     body {
             background-image: linear-gradient(to bottom,white,blue );
                     }
