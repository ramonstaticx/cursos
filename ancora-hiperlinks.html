Aprendendo sobre hiperlinks

Aprendendo um pouco sobre a tag < a > que significa 'âncora' e usando ela junto com href pode chamar uma url ou até mesmo um arquivo no diretório, um bom ex: 

                
                    <a href="http://google.com" target="_blank rel= "external""> Google</a>

E juntando com target="_blank" quando clicar no link irar abrir uma nova guia, ao invés de sair da página atual e ir para a página direcionada, e rel="external" para se referir que é uma url externa.

Porém o parametro target=self abre na mesma aba o link.

Obs: Nâo é legal por o target="_blank" para não poluir muito o navegador e ter várias páginas abertas do mesmo site

Aqui será redirecionado ao Google

Obseve que se usar target blank ele me direciona a pagina sem sair da outra q eu estava

Dica: para dizer que o caminho esta vasio usamos #

                
                    <a href="#">texto</a>

Agora vou mostra exemplo de como chamar um subi titulo da pagina

Pode se dizer que teremos de usar a tag id para darmos uma identificação para cada um. O id tem que vir depois de uma tag um bom exemplo é: 

                
                    <h2 id="qualquer nome "> texto</h2>

Logo em seguida para chamar ela como está as Lorens (sobre mim, história e trabalho), teremos de usar o seguinte Código: 

                
                    <a href="#o nome que foi do">texto</a>

Sendo mais especifico em cada Código

  • Sobre mim
  • Hitória
  • Trabalho 
                
                    <nav>
                     <ul>
                            <li><a href="#about">Sobre mim</a></li>
                            <li><a href="#history">Hitória</a> </li>
                            <li><a href="#works">Trabalho</a> </li>
                        </ul>
                </nav>

Logo em seguida as Lorens com seus sub titulos, com os id, ex de uma só o Sobre mim

                
                    <h2 id="about" style="color: black;">Sobre mim</h2>
