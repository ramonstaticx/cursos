Lista

Aqui vamos falar de listas ordenadas, listas desordenadas e listas com descrições

A lista em html é declarada pela tag <li>, mas ela tem que iniciar com <ol> ou <ul> para dizer

se ela é ordenada ou normal.

OL= a uma lista contendo item 1, item 2, item 3 e assim por diante .

UL= é uma lista comum com apenas um símbolo na frente (o mais normal é o símbolo que é um ponto.
ex:

  1. aqui uma lista ordenada
  2. podendo trocar os numeros por romanos e letras, para sinalizar a ordem da lista
  • aqui uma lista UL
  • olha que é pontos na frente

exemplo do código:

             
                        <li>texto</li>
                        <li>texto</li>

Quando usar <nav>?

Porém se for usar a tag <li>acompanhada de âncora <a>, observe que tem de ser acompanhado antes pela tag <nav>
Ex: 

            
                <nav>
                <ul>
                 <li><a href=""></a></li>
                 <li><a href=""></a></li>
                </ul>
                </nav>

Tag + Parâmetro na Lista (troca do símbolos, centralizando ...)

Para a tag ol trocamos os números acrescentando um parâmetro type que podemos no lugar de 1,2,3,... por em romano ou até mesmo letras A,b,c,..., isso em lista ORDENADA (OL) em listas não ordenadas só se usa para mudar o símbolo como ao invés de ponto por quadrado, copas,... 

            
                <ol type="I,A,a,1">
                    <li >tipo de lista </li>
                    </ol>
