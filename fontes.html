Propriedades básicas das Fontes

Agora alguns exemplos de propriedades de font

font-family:.....família (tipo) de fontes;

font-size:.......tamanho da fonte;

font-style:......estilo da fonte;

font-stretch:....grau de espansão/contração dos glifos;

font:............maneira abreviada para declarar todas as propriedades anteriores (exceto cor).

Tipografia

Tipografia das fontes é o sentimento que as fontes passam.

FONTES --> serifadas= contem serifo (um pésinho)
sans-serif= sem serifo
mono espaçada= todas as letras têm a mesma quantia de espaço.
handwriting= imita a escrita real
display fonte que não contém regras

Tamanho da Fonte

Por padrão no html a fonte tem 1em=16px, agora H1, H2, H3,... já tem as seguintes medidas.

h1 = 2em= 32px

h2= 1,5em= 24px

h3= 1.17= 18.72px

h4= 1em= 16px

h5= 0.83em= 13.28px

h6= 0.67em= 10.72

A propriedade font-size= significa Tamanho da fonte.

Importando Fontes

Usando o site Google fontes ou o do site DaFonte.com

Para importar vamos usar o css

Do google vamos usar @import, após selecionar a font, e então vamos copiar td que estiver dentro do @import vamos colar tudo dentro do css, logo em Seguida vamos pegar a familia dela logo abaixo em CSS rules to specify families, para usar essa fonte onde quisermos 

            
                @import url('https://fonts.googleapis.com/css2?family=Charis+SIL:ital,wght@0,400;0,700;1,400;1,700&display=swap');

                h1{
                    {
                        font-family:'Charis SIL', serif ;

                }
               /*--Perceba que vamos só passar a usar a familia da fonte na hr de declarar--*/

vai passar a ser assim:

            
                h1 {
                    font: normal bolder 3em 'Charis SIL', serif;
                }

Fontes externas

Fontes externas são as fontes que foram feitas downloads, ao ivés de importar

Usaremos então o @fonte-face. 

            
                @font-face {
                    font-family: ;
                    src: url();
                }

Dentro de
font-family: pode por qualquer nome para fonte ;
src: url('aqui o caminho da fonte .o formato') format ('o formato da fonte'), aqui posso repetir outra url se a fonte tiver mais de um formato;
fort-weight: normal;
fort-style: normal;

Um bom exemplo na pratica é uma fonte de coração do site: 

            
                @font-face {
                    font-family: terror;
                    src: url('../fonts/aAvocadoTaco.otf') format('opentype'), url('../fonts/aAvocadoTaco.ttf') format('truetype');
                    font-weight: normal;
                    font-style: normal;
                }

Então a fonte ficaria assim.

Obs: foi criada uma pasta fonts, e baixei as fontes la dentro.

Tipos de Formatos

  • opentype (otf) normal a que funciona em qualque navegador
  • truetype (ttf) normal funciona em quase todos navegadores
  • embedded-opentype
  • truetype-aat (apple advanced Typography)
