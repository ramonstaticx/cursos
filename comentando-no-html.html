Fazendo Comentário dentro do Código html

Como fazer um comentário dentro do html, que é usando os seguinte comandos <!-- comente aqui -->, ou leia o código para ver que o comentário feito não está aparecendo na página. 

        
            <!-- o comentário que quero fazer-->

Fazendo comentário no CSS

Para fazer um comentário dentro das CSS usamos /* para abri o comentário, e */ para fechar o comentário
