A forma Correta de se colocar uma Imagem

Os melhores formatos para se trabalhar com img é .jpeg e .png, em uma resolução 50

Para que o google reconheça melhor o site devemos otimizar o site colocando a imagem dentro das tag <picture>, e ao invés de dimensionar ela usando width, mas usar o algum editor de imagem, criar 3 tamanhos( 300x300, 700x700 e 1000x1000), para melhor adaptar em qualquer aparelho.

Esse seria o código com apenas uma imagem. 

        
            <picture>
            <img src="a origem da imagem" alt="sempre o'que a imagen significa">
            </picture>

Agora vamos ao código que realmente vai fazer a diferença, e repare que acrescentamos media=com 50px a mais em cada imagem para que na hora q diminuir o navegador não dê um pulo e sim suavize.

E tem a seguinte ordem para seguir, da menor para a maior e sempre a última é a maior e não contém a tag media, observe o código abaixo. 

        
            <picture>
            <source media="(max-width: 750px)" srcset="foto pequena" type="image/png">
            <source media="(max-width: 1050)" srcset="foto média" type="image/png">
            <img src="o arquivo da imagen" alt="descrição da imagen">
            </picture>

Observe que todos finais em relação a ao parametro media ou download, é final type, e por isso para saber o parametro type do arquivo que estamos pondo usamos o site.

iana.org

Centralizando a Imagem

Para centralizar a imagem usamos o CSS declarando propriedade margin com o valor auto 

                
                    div->(que contem a imagen)
                    div {
                        margin: auto;
                    }

                    /* ou usar o tradiciona text-aling */

                    div {
                        text-aling: center;
                    }
