Indentificadores

Aqui vamos ver os id e class

Mas antes temos de saber algumas regras importantes que que faz com que a class seja diferente da id. O id= só pode ser usado uma unica vez no mesmo elemento(tag) dentro do documento html, enquanto o class= pode ser usada varias vezes,e no Css a no css a class= vem antes do id=

Um bom jeito de se explicar o id= é que ex:

        
            <h1 id= "principal">texto</h1>

Por o h1 ter recebido id o valor principal(podendo ser qualquer nome) no mesmo documento não poderá ter nenhuma id com o mesmo valor, nem que seja para expecificar novamente a mesma tag. Porem pode a mesma tag ter um id com valor diferente.

id= indentificador

class= classe

Como é declarado o id

no html id=

no css #Nome-Atribuido (sempre em minusculo)

Como é declarado a class

no html class=

no css .Nome-Atribuido (sempre minusculo)

no html mais de uma class juntas class="valor1 valor2

        
            <h1 class="principal">texto</h1>
            <!--ou duas class juntas-->
            <h1 class="principal secudario">texto</h1>

Pseudo class

Uma pseudo-classe CSS é uma palavra-chave adicionada às declarações de um seletor após um sinal de dois pontos ( : ) e especificam um estado especial de um elemento. Existem várias pseudo-classes para estilos, podemos citar :hover, :visited, :active, :checked, :empty e :focus

A pseudo-classe é colocada da seguinte forma primeiro a div (tag ou class) depois : e então um seletor.

            
                a:hover{
                    color: aqua;
                }

Desta forma toda vez que passar o mouse por cima vai ficar da cor aqua(azul bem claro).

Vou usar o texto a baixo para usar o :hover de uma forma que mostre outro texto, e logo em seguida vou por todo o codigo abaixo:

VEJA A MAGICA, PASSE O MOUSE POR CIMA

A MAGICA ACONTECEU 

            
                /* no html*/
                <div class="pseudo-texto"> VEJA A MAGICA, PASSE O MOUSE POR CIMA
                    <p>A MAGICA ACONTECEU </p>
             </div>

                   /*no CSS*/
                   div.pseudo-texto{
                    font-family: terror;
                    text-align: center;
                    color:yellowgreen
                }
                div.pseudo-texto>p{
                    display: none;
                }
                div.pseudo-texto:hover>p{
                    display: block;
                }

Algumas das pseudo class e oque fazem:

a:visited{color: darkred}
✎ link depois de visitado fica vermelho escuro
a:hover{color:black; text-decoration:underline}
✎ ao passar o mouse por cima ficará com sublinhado de preto
a:active{color:yellow}
✎fica amarelo bem na hora de clicado

Pseudo Elementos

Um pseudo-elemento CSS é uma palavra-chave adicionada às declarações de um seletor após dois sinais de dois pontos ( :: ) e permitem que você formate um pedaço específico do elemento referenciado. Os principais pseudo-elementos usados nas CSS são ::before, ::after, ::first-letter, ::first-line.

.especial ::before {content: '✍';}
psdeudo elemento declaração e valor class criada no html

alguns exemplos e as explicaçôes:

        
            .pseudo-element{
                font-size: 1.5em;
                font-weight:bold ;
                text-align: center;
            }
            .pseudo-element::before{
                content: '✍' ;
               /*simbolo antes do txt*/
            }
            .pseudo-element::after{
                content: '✍';
                /*simbolo depois do txt*/
            }
veja o código a cima

OBSERVE QUE NÃO PODE FALTAR AS ASPAS SIMPLES DENTRO DO content: 'simbolo'

Pai e Filho

Vimos logo a cima que div>p isso significa que a p é filho da div, assim fica mais facil para declarar que o paragrafo dentro da quela div vai ter certo valores, mais para isso ela teria de ser uma class ou id, para não ser confundidas com as outras iguais ou parecidas

        
            no html 
            <div class="p-da-div">
            <p>texto</p>
            </div>

            no Css
            .p-da-div>p {
                /*e aqui dentro a declaração*/
            }
