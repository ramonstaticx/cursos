FORMATAÇÃO NO HTML

No html usamos algumas tags para melhor formatar a página, como usar a tag <h1> é mais usada para por títulos e <h2 >para subtítulo, e os mais conhecidos vai até h6.

Usamos também para parágrafos a tag p

TAG <p> e <br>

Para sinalizar que é um parágrafo só e nada mais, usamos a tag <p> mas ela é uma tag que abre e tem que ser fechada, e após fechada automaticamente ela pula para a próxima linha no navegador ex: 

                
                    <p> texto</p>

A tag <br> é usada para simplesmente pular linha e não tem que fechar

HTML e CSS trabalhando juntos

Para melhor formatação usamos junto das tags o css, para por exemplo dar melhor margem ao parágrafo ou cor aos títulos e subtítulos, centralizar o corpo do parágrafo ou simplesmente alinhar todo o texto. Alguns exemplos abaixo. 

            
                ------------------
                Aqui o corpo de toda a pagina
                body {
                    text-align: center;
                    margin: 8px;
                }
                ------------------
                Aqui oque eu quero para quando eu usar a tag p 
                p {
                    margin: 8px;
                    text-align: justify;
                    text-indent: 20pt;
                }
                ------------------
                aqui atributos para os titulos e subtitulos
                h1 {
                    color: red;
                }
                
                h2 {
                    color: blue
                }
                ------------------

Tags strong, em ou i

As Tags B(negrito) foi subistituida pela tag STRONG, e a tag I (itálico) substituidas pela tag EM (ênfase)

A tag <strong> </strong> da importância(negrito) a palavra, e deixando em negrito

A tag <i> </i> deixa a palavra em itálico

A tag <em> deixa em itálico e da importância a palavra

Centralizando texto e img

Eu uso a tag + o parâmetro (style) junto depois a propriedade (text-align) e logo em seguida o valor (center) para escolher a posição, que eu quero na página.

e uso junto a tag style seguida de color para por a cor .

            
                    <div style="color: blue;text-align: center;">Centralizando texto e img </div>
