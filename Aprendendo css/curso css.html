AULA DE CSS

O que é CSS

CSS é uma maneira de estilizar a pagina do html, porque no momento do curso estou usando HTML e vou passar estilizar com o css

Regrinhas a seguir

  • Sempre nomear os arquivos com final .css ou .html ex: ( index.html ou style.css) mesmo podendo usar o css dentro do html
    • obs: no vscode o preview só se ve o html e não se css se estiver em link.
  • CSS sem haverá comando usando { } ex: h1{ color: blue }

DICAS

  • Para estilizar melhor as fontes pegar, usar fontes disponíveis no site usando o site fontes do google
  • Porem é indicado a por a fonte direto ao inves de por o link, porque ai o site demora mais para carregar.
  • Muito cuidado com a formatação do CSS, ex: <a href="https:google.com">, e com isso acabar juntando o "a com href ", e causar um erro de formatação.
  • DEVTOOL, é tudo dentro do F12 no browser
  • Propriedade tem valor.

    Value pode ser chamado de Data type

VEDOR PREFIXES(PREFIXOS).

Permite que browser adicione 'features' (novidades) a fim de colocar em uso alguma novidade que vemos no CSS.


Ex:

-webkit-background: text; /*chorme, safari, ios e android */

-moz-background: text; /* momzilla(firefox) */

-ms-background: text; /*internet Explorer*/

-o-background: text; /*opera*/
