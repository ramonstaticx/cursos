Tipografia

Tipografia das fontes é o sentimento que as fontes passam.

FONTES --> serifadas= contem serifo (um pésinho)
sans-serif= sem serifo
mono espaçada= todas as letras tem a mesma quantia de espaço
handwriting= imita a escritra real
display fonte que não contem regras

Dica: Usar sans-serif em sites com textos longos.

            
                body{
                    font-famly:arial
                }

font-family: arial;

corpo do html modelo da fonte qual fonte vai ser

Obsserve a propriedade font-family: que signifaca familia da fonte ou seja vai ser colocado o modelo da fonte e mais umas 3 da familia para que se o computador não tiver a fonte vai ler a seguinte, ex: font-family: Arial, Helvetica, sans-serif;
